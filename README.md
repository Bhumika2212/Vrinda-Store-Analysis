# Vrinda-Store-Analysis
Data analysis using Excel dashboards
Excel Data Analysis Project

Project Overview
This project is based on data analysis using Microsoft Excel. The main aim of this project is to analyze the dataset and extract useful insights using Excel tools like charts, pivot tables, and dashboards.

Dataset Used
The dataset used in this project contains data related to Sales.

Example:
- Sales data of a company
- Customer purchase data
- Student performance data

Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Line, etc.)
- Data Cleaning
- Dashboard Creation

Key Insights
- Women's are more likely to buy compared to men (~65%).
- Maharashtra, Karnataka and Uttar Pradesh are the top 3 states (~35%).
- Adult age group (20-49 yrs.) is maximum contributing (~50%).
- Amazon, Flipkart and Myntra channels are max contributing (~80%).

Conclusion
To imporve Vrinda Store Sales must target women customer of age group (20-49 yrs.) living in Maharashtra, Karnataka and Uttar Pradesh by showing ads/offers/coupons available on Amazon, Flipkart and Myntra.
